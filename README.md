# Antigravity Project

> **极客思维 · 冷酷理性 · 架构审计**

本项目是一个基于「硅基战略局」架构审计准则驱动的高标准开发工作空间。

## 核心规约
本项目受 `.agents/` 目录下的规约集双重约束：
- **[启发式准则](.agents/heuristics/core_heuristics.md)**: 底层决策与执行的哲学指导。
- **[治理规则](.agents/rules/core_rules.md)**: 身份定义、开发规范与运维手册。

## 快速开始

### 1. 环境初始化
```bash
# 生成本地环境变量
cp .env.example .env
# 请手动编辑 .env 填充必要的 API Key 与 数据库配置
```

### 2. 目录概览
- `src/`: 核心业务源代码。
- `.agents/`: AI 助手的治理核心（准则、规则、工作流）。

## 协作模式
1. **导演式反馈**: CEO 下达目标，CSO (AI) 提供执行路径并进行架构审计。
2. **强制前置审计**: 所有核心变更均需通过 `.agents/rules` 的符合性检查。
