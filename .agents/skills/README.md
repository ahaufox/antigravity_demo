# AI 助手技能库 (Skills Library)

本目录存储 AI 助手的核心技能定义，用于指导不同场景下的专业任务执行。

## 📁 目录索引

### 🎨 前端开发
- [MindSeed Engine 开发 (MindSeed Engine Dev)](mindseed-engine-dev/SKILL.md): 指导 `scripts/ui/mindseed-engine` 下 Astro/React 项目的开发与维护。
- [UI 设计 (UI Design)](ui-design/SKILL.md): 指导高保真、可交互的前端界面设计与实现。

### ⚙️ 后端开发
- [MindSeed 内容生成 (MindSeed Content Generation)](mindseed-generation/SKILL.md): 指导 MindSeed 内容类型从 Schema 定义到 HTML 模板生成的全流程。
- [API 测试生成 (API Test Generation)](api-test-generation/SKILL.md): 指导使用 FastAPI TestClient 和标准化 Mock 方案生成高质量接口测试。

### 🧪 测试与开发辅助
- [测试审查与补全 (Test Review)](test-review/SKILL.md): 系统性审查测试文件、智能补全测试用例、对比PRD文档一致性。
- [路径重构 (Path Refactor)](path-refactor/SKILL.md): 检查并重构 Python 脚本文件路径定义，确保正确导入后端组件或核心模块。

### 🔧 通用技能
- [Git 自动化套件 (Git Auto Suite)](git-auto-suite/SKILL.md): 极速安全的代码变更分析、符合 CC 规范的提交与推送。
- [代码评审 (Code Review)](code-review/SKILL.md): 统一的代码质量检查标准。
- [内容起草 (Content Drafting)](content-drafting/SKILL.md): 通用文案与文档撰写技能。
- [MCP 开发 (MCP Development)](mcp-development/SKILL.md): Model Context Protocol 工具开发规范。
- [中文本地化 (Chinese Local)](chinese-local/SKILL.md): 确保所有代码注释、文档字符串和描述均使用中文。

---

> [!TIP]
> 每个技能文件均包含 `角色` (Role) 相关的定义模块与 `Negative Constraints` (负向约束)，使用前请仔细阅读。
