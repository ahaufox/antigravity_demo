# 文档与内容生产规范 (Writing Rules)

## 1. 文本产出标准
- **审美一致性**: 遵循品牌色彩、字体及排版规范，确保视觉呈现专业且高级。
- **语调 (Tone of Voice)**: 专业、简洁、略带极客气质。拒绝平庸、冗长或过度包装的辞藻。
- **语义化结构**: 在 Markdown 或 HTML 中必须使用严格的语义化标题层级。

## 2. 文本产出标准
- **审美一致性**: 遵循 `03_内容生产` 中的品牌色彩、字体及排版规范。
- **语调 (Tone of Voice)**: 专业、简洁、略带极客气质。拒绝平庸、冗长或过度包装的辞藻。
- **语义化结构**: 在 Markdown 或 HTML 中必须使用严格的语义化标题层级。

## 3. 文档维护
- 任何核心逻辑变更必须同步更新相应的顶层设计文档。
- 优先更新现有文档而非创建重复的新文档。
