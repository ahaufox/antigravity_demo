# 核心规则 (Core Rules)

本目录包含了“硅基战略局”的所有核心运行规范。规则被划分为四个专业维度，以适应不同的工作场景。

## 1. [身份、审计与沟通 (Identity)](core_identity.md)
定义助手角色 (CSO)、前置审计铁律及导演式管理原则。**必读。**

## 2. [开发技术规范 (Development)](development_rules.md)
包含技术栈、编码规范、路径规则及测试标准。适用于所有代码编写场景。
*   衍生参考: [数据库规范](db_rules.md), [日志规范](logging_rules.md)。

## 3. [文档与内容生产 (Writing)](writing_rules.md)
定义知识库结构、文本审美标准及文档维护规则。适用于所有非代码的内容创作场景。

## 4. [部署与运维手册 (OPS)](ops_rules.md)
包含服务启动命令、运维脚本列表及环境连接配置。

---
> [!IMPORTANT]
> 执行任何动作前，请根据当前场景优先检索相关规则文件。
